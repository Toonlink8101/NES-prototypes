DPCM Driven PCM - Uses the IRQ of the DPCM channel to drive software-driven audio that is both polyphonic and multitimbral. Will eventually also drive visual effects simultaneously.

Stableframe Transcription - Converts [this project](https://github.com/tcr/8bit-demos/tree/main/nes/stableframe) from AS Macroassembly into ca65 assembly. It uses the DPCM channel to drive visual effcts, namely multiple instances of midline scrolling.
